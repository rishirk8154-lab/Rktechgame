Python Programming Course - Introduction

What is Python?

Python ek popular high-level programming language hai jo easy to learn aur use karne mein simple hai. Ye interpreted language hai, matlab aapko code run karne ke liye alag se compile karne ki zarurat nahi hoti. Python web development, data science, machine learning, automation, scripting aur kai aur areas mein use hota hai.

Features of Python

  • Easy to read and write
  • Extensive standard library
  • Supports multiple programming paradigms: procedural, object-oriented, functional
  • Dynamic typing
  • Large supportive community

Installing Python

Python official website python.org/downloads se download karein. Windows, Mac, Linux ke liye installers available hain. Aap apni pasand ka IDE choose kar sakte hain jaise IDLE, PyCharm, Visual Studio Code, etc.

Running Your First Python Program

Python code likhne ke liye aap simple text editor ya IDE ka use kar sakte hain. Yahan ek sabse basic program hai jo "Hello, World!" print karta hai:

print("Hello, World!")

Is program ko run karne ke liye:

  1. Ek file create karein jaise hello.py
  2. Upar wala code likhein aur file save karein.
  3. Command prompt/terminal kholen, us folder me jayein jahan file saved hai.
  4. Type karein python hello.py (ya Mac/Linux me python3 hello.py).
