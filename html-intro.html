HTML ka Parichay

HTML ka matlab hai Hyper Text Markup Language. Ye ek standard markup language hai jiska use web pages banane ke liye hota hai.

HTML kya hai?

HTML ek programming language nahi, balki ek markup language hai jisme tags ka use karke webpage ka structure aur content banaya jata hai. Ye web pages ka backbone hoti hai jisme headings, paragraphs, images, links, aur bahut kuch define hota hai.

HTML ki Key Features

  • HyperText: Iska matlab hai ki aap web pages me easily links bana sakte hain jo dusre pages ya location par le jaate hain.
  • Markup Language: Tags ka use karke content ko markup karna jaisa <h1>, <p> etc.
  • Structure: Webpage ka ek hierarchical structure provide karta hai.
  • Semantics: HTML5 me naye semantic tags aaye hain jaise <header>, <footer>, <article> jo accessibility aur SEO ke liye important hain.

Basic HTML Page Structure

Har HTML document ka structure kuch iss tarah hota hai:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Mera Pehla Webpage</title>
</head>
<body>
  <h1>Welcome to HTML</h1>
  <p>Ye mera pehla webpage hai jo HTML se bana hai.</p>
</body>
</html>

Apna Pehla HTML Page Kaise banayein?

1. Koi text editor kholen (jaise Notepad, VS Code, Sublime Text).
2. Upar diya gaya basic HTML structure ko copy karke nayi file me paste karein.
3. File ko .html extension ke saath save karein, jaise index.html.
4. Is file ko apne browser me kholen aur apna pehla webpage dekhein.
