Step-by-Step Guide: Setting Up JDK and IDEs

1. What is JDK?

JDK (Java Development Kit) Java applications banane ke liye zaroori tools ka set hai, jisme compiler, java runtime, libraries aur debugging tools shamil hain.

2. Downloading and Installing JDK

  1. Apne browser me Oracle Java Downloads ki official website kholen.
  2. Windows, Mac, ya Linux ke hisab se sahi JDK version download karein.
  3. Downloaded installer ko run karein aur installation prompts follow karein.
  4. Installation complete hone ke baad, JDK ka install location dhyaan se note karein.

3. Setting Environment Variables (Windows)

  1. Windows Start menu me "Environment Variables" search karein aur "Edit the system environment variables" kholen.
  2. "Environment Variables" button par click karein.
  3. "System variables" me "New" par click karein, name me JAVA_HOME dalein aur value me JDK ka installation path (jaise C:\Program Files\Java\jdk-XX.X.X) daliye.
  4. System variable `Path` ko select karein, "Edit" par click karein, aur naya entry add karein: %JAVA_HOME%\bin.
  5. Sab OK kar ke windows close karein.
  6. Command Prompt open karke java -version aur javac -version command chalakar verify karein ki JDK sahi se install hua hai.

4. Setting Environment Variables (Mac/Linux)

  1. Terminal kholen.
  2. Apne shell configuration file me (jaise ~/.bash_profile, ~/.zshrc) niche lines add karein: 
    export JAVA_HOME=$(/usr/libexec/java_home)
export PATH=$JAVA_HOME/bin:$PATH
  3. File save kar ke terminal ko restart karein ya source ~/.bash_profile chalayein.
  4. java -version command chalakar installation verify karein.

5. Installing IDEs

JDK install karne ke baad, aap apne pasand ki IDE install kar sakte hain. Kuch popular IDEs hain:

  • IntelliJ IDEA: Powerful Java IDE with smart coding assistance. Download at jetbrains.com
  • Eclipse IDE: Open source IDE widely used for Java development. Visit eclipse.org
  • NetBeans IDE: Another popular open-source IDE by Apache. Available at netbeans.apache.org

Installation steps har IDE me thode alag ho sakte hain, par generally downloaded installer ko run karke standard prompts ko follow karna hota hai.

6. Verifying Setup and Creating First Java Program

  1. IDE open kar ke naye Java project create karein.
  2. Simple HelloWorld.java class banayein: 
    public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
  3. Program run karke output verify karein.
