CSS Fundamentals

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It controls the layout, colors, fonts, spacing, and overall visual appearance of web pages. CSS enables separation of content from design, making web pages easier to maintain and aesthetically pleasing.

CSS Syntax

A CSS rule set consists of a selector and a declaration block:

selector {
  property: value;
  property2: value2;
}

Example:

h1 {
  color: blue;
  font-size: 24px;
}

CSS Selectors

Select elements to apply styles to:

  • Element Selector: Targets all elements of a type. (e.g., p {})
  • Class Selector: Targets elements with a class attribute, denoted by a dot. (e.g., .menu {})
  • ID Selector: Targets a unique element with an ID, denoted by a hash. (e.g., #header {})
  • Attribute Selector: Targets elements with specific attributes. (e.g., a[href^="https"] {})
  • Descendant Selector: Targets elements inside another element. (e.g., nav ul li {})

CSS Properties

Common CSS properties include:

PropertyDescription
colorText color
background-colorBackground color of elements
font-sizeSize of text
marginSpace outside an element's border
paddingSpace inside an element's border
borderBorder around an element
width, heightDimensions of an element
displayHow an element is displayed (block, inline, etc.)

The CSS Box Model

Every HTML element is considered a rectangular box and CSS controls its dimensions through the box model, which consists of:

  • Content: The actual content like text or images.
  • Padding: Space between content and border.
  • Border: Surrounds the padding and content.
  • Margin: Space outside the border, separating elements.

Why Use CSS?

CSS allows you to separate content and presentation, reuse styles efficiently across your site, and create responsive layouts for different devices. It provides control over typography, color schemes, layout, and interactive effects.

Summary

Understanding CSS fundamentals is essential for designing beautiful and functional web pages. Practice writing CSS rules, experimenting with selectors and box model to gain expertise.

CSS Kya Hai?

CSS (Cascading Style Sheets) ek stylesheet language hai jo HTML ya XML document ke presentation ko control karti hai. Yeh web page ke layout, rang, fonts, spacing aur poore visual appearance ko define karti hai. CSS content aur design ko alag karta hai jisse website maintain karna aasan aur acchi dikhti hai.

CSS Syntax

CSS rule set mein selector aur declaration block hota hai:

selector {
  property: value;
  property2: value2;
}

Udaharan:

h1 {
  color: blue;
  font-size: 24px;
}

CSS Selectors

Elements ko style dene ke liye selectors ka istamaal hota hai:

  • Element Selector: Sabhi elements ko target karta hai. (Jaise p {})
  • Class Selector: Un elements ko jinke paas class attribute hota hai target karta hai, dot ke sath likha jata hai. (Jaise .menu {})
  • ID Selector: Unique element jiska ID hota hai, hash ke sath likha jata hai. (Jaise #header {})
  • Attribute Selector: Un elements ko target karta hai jinke paas specific attribute hota hai. (Jaise a[href^="https"] {})
  • Descendant Selector: Jo elements doosre element ke andar hote hain unko target karta hai. (Jaise nav ul li {})

CSS Properties

Kuch common CSS properties:

PropertyDescription
colorText ka rang
background-colorElement ki background ka rang
font-sizeText ka size
marginElement ki border ke bahar ki jagah
paddingElement ke andar content aur border ke beech ki jagah
borderElement ke charo taraf border
width, heightElement ki lambai aur chaudai
displayElement ka display style (block, inline, etc.)

CSS Box Model

Har HTML element ek rectangle box hota hai jiska size aur space CSS ke box model se control hota hai, jis mein shamil hain:

  • Content: Actual text ya image jo dikhaya jata hai.
  • Padding: Content aur border ke beech ki jagah.
  • Border: Padding aur content ke chaaro taraf border.
  • Margin: Border ke bahar ki jagah jo elements ke beech gap deti hai.

CSS Kyun Use Karein?

CSS se aap content aur presentation ko alag kar sakte hain, jo styles ko baar baar reuse karna aasan banata hai, aur alag-alag devices ke liye responsive layout create karta hai. Yeh typography, color scheme, layout aur interactivity ko control karta hai.

Samapan

CSS ke moolbhoot tatvon ko samajhna zaroori hai achhi aur functional web pages banane ke liye. CSS likhne ki practice karein, different selectors aur box model ko samjhein taaki aap expert ban saken.
