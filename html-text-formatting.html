HTML me Text Formatting

HTML me kuch special tags hote hain jinhe use karke hum text ko alag alag style de sakte hain jaise bold, italic, underline, highlight, aur bhi bahut kuch.

Common Text Formatting Tags

  • <b> : Isse text ko bold banate hain.
  • <i> : Isse text ko italic style milti hai.
  • <u> : Text ko underline karta hai.
  • <mark> : Text ko highlight karta hai, matlab background color deta hai.
  • <sup> : Superscript yani text ko upar chhota dikhata hai (jaise footnotes).
  • <sub> : Subscript yani text ko niche chhota dikhata hai (jaise chemical formulas).
  • <small> : Text ko chhota dikhata hai.
  • <del> : Deleted text ko dikhaata hai, generally is par line strike karti hai.
  • <ins> : Inserted text ke liye underline karta hai.

Example:

<p>Yeh normal text hai.</p>
<p><b>Yeh bold text hai.</b></p>
<p><i>Yeh italic text hai.</i></p>
<p><u>Yeh underlined text hai.</u></p>
<p><mark>Yeh highlighted text hai.</mark></p>
<p>Chemical formula: H<sub>2</sub>O</p>
<p>Footnote: x<sup>2</sup></p>
<p><small>Yeh chhota text hai.</small></p>
<p><del>Yeh deleted text hai.</del></p>
<p><ins>Yeh inserted text hai.</ins></p>
