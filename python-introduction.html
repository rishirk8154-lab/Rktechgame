Python Introduction

Python ek bahut popular high-level programming language hai, jise 1991 mein Guido van Rossum ne banaya tha. Yeh language apni simple aur readable syntax ke liye famous hai, isliye naye programmers ke liye bahut acchi maani jaati hai.

Python kis liye use hota hai?

  • Web Development (website banane ke liye)
  • Data Analysis aur Visualization
  • Artificial Intelligence (AI) aur Machine Learning
  • Scientific Computing
  • Automation aur scripting
  • Game Development
  • Network Programming

Python install kaise kare?

Python apne computer par install karne ke liye official website pe jaayein: python.org aur latest version download karke install karein.

Sabse pehla Python program

Chaliye ek simple program likhte hain jo screen par "Hello, World!" print kare:

print("Hello, World!")

Is program ka output yeh hoga:

Hello, World!

Ismein print() function use kiya gaya jo text ko screen par dikhata hai.

Python ki basic cheezein

1. Variables aur Data Types

Variables ko hum data store karne ke liye use karte hain. Python automatically data type samajh leta hai:

name = "Ankit"      # string
age = 21             # integer
height = 5.8         # float
is_student = True    # boolean

2. Comments

Comments code ko explain karne ke liye use hote hain, yeh run nahi hote:

# Yeh ek single-line comment hai

"""
Yeh
multi-line
comment hai
"""

3. Control Flow (If-else)

Code ke blocks ko indent karke define karte hain:

if age >= 18:
    print("Adult")
else:
    print("Minor")

4. Loops

Koi kaam ko baar-baar karne ke liye loops use hote hain:

for i in range(5):
    print(i)

Conclusion

Python apni simple aur powerful syntax ke wajah se beginners ke liye sabse acchi language hai. Bahut saare use cases aur community support ke saath, yeh programming seekhne ke liye best choice hai.

Aap aage chal kar functions, modules, object-oriented programming, file handling jaise topics bhi seekh sakte hain.
