HTML Tables Basics

Tables ka use data ko rows aur columns me arrange karne ke liye hota hai. Websites par data ko structured format me dikhane ke liye tables bahut kaam aate hain.

HTML Table banane ke liye essential tags

  • <table> : Table create karta hai. Ye tag table ka container hota hai.
  • <tr> : Table row define karta hai.
  • <th> : Table header cell create karta hai. Ye by default bold aur center aligned hota hai.
  • <td> : Table data cell create karta hai. Ye headers ke neeche actual data hota hai.

Basic Table Structure Example

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>Country</th>
  </tr>
  <tr>
    <td>Rahul</td>
    <td>26</td>
    <td>India</td>
  </tr>
  <tr>
    <td>Anita</td>
    <td>22</td>
    <td>Nepal</td>
  </tr>
</table>

Table Output :

Name Age Country
Rahul 26 India
Anita 22 Nepal

Additional Table Tags:

  • <thead> : Table ke header section ko define karta hai.
  • <tbody> : Table ke main content section ko define karta hai.
  • <tfoot> : Table ke footer section ko define karta hai.

Rowspan aur Colspan Attributes

Table cells ko multiple rows ya columns me span karne ke liye ye attributes use hote hain:

<td rowspan="2">Ye cell do rows me span karega.</td>
<td colspan="3">Ye cell teen columns me span karega.</td>

Ye especially complex tables banane me kaam aate hain.
